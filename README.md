# Safe Step
A full-stack interactive training Single Page Application (SPA) for emergency response.

## Technology Stack
- **Frontend**: Next.js 14
- **Backend**: Laravel 11 
- **Database**: PostgreSQL

## Quick Start (Docker)
**Clone the project** and ensure the sub-folders are named `safe-step-backend` and `safe-step-frontend`.
### For Windows Users
Double-click `setup.bat` or run:
```batch
.\setup.bat
```
### For Linux/Mac Users
```
chmod +x setup.sh
./setup.sh
```
